Glass Morphism Utilities

Four distinct glassmorphism effects for different use cases. Hover over each to see the interactive glow effect:

Subtle Glass

Minimal blur for subtle backgrounds

Use for: Secondary content areas

rgba(255, 255, 255, 0.04)

Standard Glass

Balanced glassmorphism effect

Use for: Cards, panels, modals

rgba(255, 255, 255, 0.08)

Strong Glass

Enhanced blur with stronger borders

Use for: Navigation, headers

rgba(255, 255, 255, 0.12)

Intense Glass

Maximum blur for dramatic effect

Use for: Hero sections, overlays

rgba(255, 255, 255, 0.16)

Typography & Semantic Elements

REM Usage Audit

All typography uses rem for sizing. This maintains accessibility for users who change their browser font size. The body element has a size of 1.15rem which equates to 18.4px for paragraph text, following Simple.css sizing approach.

Heading 1 3rem

Heading 2 2.6rem

Heading 3 2rem

Heading 4 1.44rem

Heading 5 1.15rem
Heading 6 0.96rem

Paragraph text: 1.15rem (18.4px) - This matches Simple.css sizing for optimal readability and accessibility compliance.
